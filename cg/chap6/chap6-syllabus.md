# Unit 6: Graphical Standards (4 Hrs)

### Specific Objectives:
Gain a good understanding of the use of graphical standards necessary for achieving software portability. Become familiar with the basic structure or format of a graphical file and know the ways of visually representing data sets. Learn about writing graphical programs that can be ported to multiple hardware platforms without extensive rewriting of the programs through the use of graphical standards.

### Contents:
- 6.1 Need for Machine Independent Graphical Languages
- 6.2 Graphical Standards: PHIGS, GKS
- 6.3 Graphics Software Standards and Language Binding
- 6.4 Overview of Graphics File Formats
- 6.5 Visualization of Data Sets
