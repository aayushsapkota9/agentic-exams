# Unit 2: Scan Conversion Algorithms (7 hrs)

### Specific Objectives:
Learn about various scan conversion and area filling algorithms. Students will be able to implement these scan conversion algorithms using a programming language. This chapter helps students learn how algorithms are used to generate output primitives and fill them with specified intensities.

### Contents:
- 2.1 Scan Conversion
- 2.2 Line Drawing Algorithms
  - 2.2.1 Digital Differential Analyzer
  - 2.3.2 Bresenham's Algorithm
- 2.3 Circle Generation Algorithm
- 2.4 Ellipse Generation Algorithm
- 2.5 Filled Area Primitives
  - 2.5.1 Scan Line Polygon Fill Algorithm
  - 2.5.2 Boundary Fill Algorithm, Flood Fill Algorithm
