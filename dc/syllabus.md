## 1. Course Description
Data Communication is a fundamental aspect of computer engineering and networking. This course provides students with a comprehensive understanding of the principles, protocols, and technologies used to transmit data between computers, devices, and networks. It explores the theoretical and practical aspects of data communication.

## 2. General Objectives
- To acquire the knowledge and concept of Data Communication.
- To recognize the requirements of digital devices to exchange data.
- To realize the basics of switching and networking.

## 3. Methods of Instruction
- Lecture, and practical.

## 4. Content in Detail

| Unit | Topics | Duration |
| :--- | :--- | :---: |
| **Unit 1: Introduction** | - Evolution of Data Communication systems<br>- Analog and Digital Data Transmission, Data Communication Terminology<br>- Standards Organizations, Applications | 2 hrs |
| **Unit 2: Data Transmission** | - Data Transmission Techniques - Parallel Transmission, Serial Transmission (Synchronous, Asynchronous and Isochronous Communication), Modes of Data Transmission<br>- Line Configuration, Bit Rate/ Baud rate, Transmission Channel, Data Rate Limits - Shannon Capacity Theorem and Nyquist Bit Rate, RS-232C (DTE-DCE, DTE-DTE) | 3 hrs |
| **Unit 3: Signals and Systems** | - Signals and their classification: Periodic and non-periodic signals; Deterministic and Random signals; Energy and Power signals; Continuous and Discrete time signals<br>- Basic Elementary Signals - Unit Step Signal, Ramp Signal, Impulse Signal, Sinusoidal Signal, Signum Signal<br>- System - Continuous and Discrete time system<br>- Basic system properties: Linearity, Causality, Stability, Static & Dynamic, and Time Invariance, Introduction to LTI System | 6 hrs |
| **Unit 4: Networking & Protocols** | - Network Types, Topology<br>- OSI layers and Functions, TCP/IP layer, Local Area Networks (LAN) Architecture, LLC/MAC & Routing<br>- IEEE Standards, Ethernet (Aloha, CSMA), Wide Area Networks (WAN): X.25, Frame Relay, ATM | 4 hrs |
| **Unit 5: Transmission Media** | - Electromagnetic Spectrum for Telecommunication<br>- Type of Propagation<br>- Guided Transmission Media: Twisted Pair Cable, Coaxial Cable, Optical Fiber, Characteristics of Unguided Communication Bands, Antennas<br>- Unguided Transmission Media: Terrestrial Microwave, Satellite Communication, VSAT, and Cellular Telephony | 5 hrs |
| **Unit 6: Impairments & Compression** | - Attenuation & Distortion, Delay Distortion, Noise & Types, interference, crosstalk<br>- Types of error & its Detection and Correction Methods<br>- Data Compression, Lossless Compression - Run Length Coding, Dictionary Coding and Huffman Coding<br>- Lossy Compression - Predictive Coding and Transform Coding | 6 hrs |
| **Unit 7: Data Link Control** | - Framing<br>- Flow Control: Stop - & - Wait, Sliding Window, Error Control: Automatic Repeat Request (ARQ), Stop-and Wait ARQ, Sliding Window (ARQ)<br>- HDLC protocol<br>- Point-to-Point protocol | 5 hrs |
| **Unit 8: Multiplexing & Switching** | - Multiplexing types and Application<br>- Multiplexing Vs Non-Multiplexing<br>- The Telephone System: Analog services and its Hierarchy<br>- Digital services and Hierarchy Circuit Switching, Packet Switching, Message Switching, and Private Branch Exchange | 5 hrs |
| **Unit 9: Encoding & Modulation** | - Line coding – Unipolar, Polar and Bipolar signaling<br>- Digital Modulation Techniques - Amplitude, Frequency, and Phase Shift Keying<br>- Analog to Digital Conversion - Pulse Code and Delta Modulation<br>- Analog Modulation Techniques - Amplitude, Frequency, and Phase Modulation<br>- Multilevel Modulation - QPSK, QAM<br>- Introduction to Modem | 9 hrs |
